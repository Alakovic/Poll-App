# PollApp

PollApp is a real-time polling application built with **Angular** that allows users to create, share, and analyze surveys instantly.  
The application uses **Supabase** as a backend service for authentication, real-time data synchronization, and database management.

## 🚀 Features

- Create custom polls with multiple answer options
- Share polls with other users via link
- Vote in real time with instant result updates
- View poll analytics and results visualization
- Responsive and modern UI

## 📊 Future Improvements

- Secure user authentication (Sign up / Login)
- Anonymous voting mode
- Comments on polls
- Advanced analytics dashboard

## 🛠️ Tech Stack

- **Frontend:** Angular
- **Backend / BaaS:** Supabase
- **Database:** PostgreSQL (via Supabase)
- **Authentication:** Supabase Auth
- **Real-time updates:** Supabase Realtime
